Customer Credit Limit

This plugin use for check Customer Credit Limit and notify to sales manager.

Customer Credit limit in general refers to the maximum amount of credit that a financial institution can extend to a prospective customer which can then be leveraged by an underwriter while funding an application. ... Using this limit, applications can be funded up to the credit limit amount allowed.

Key Highlights

  • Allowed user can set Credit limit for customer.
  • While confirming Quatation/Order if customer's credit limit Exceeded this window will displaying with Credit limit information.
  • Credit limit exceeded for customers while confirming sale order, mail will send to Sales mangers for Credit limit approval.
  • After confirming sale and credit limit exceeded then that customer will go in Credit limit hold.
  • New menu for Credit limit orders.

  • Overview
