Odoo App for Website Ecommerce Order Customer Comment/Message

Website Shop Sales Order Customer Comment/Message

This app allows your customer to send Comment/Message from your website ecommerce shop while doing shopping which will be store on backend sales order form as shown in screenshots below.


Main Features:

  • This module allows your customer to send comment/message from website shop ecommerce.

  • Customer can send the message from your website shop from below pages.

    • Cart Page.
    • Payment Page.

  • Sales team can see a customer message on sale order in the backend.

  • For more details please see below details and watch the video. If you want to show comment on sales order pdf then you can check app https://apps.odoo.com/apps/modules/14.0/website_order_customer_comment_pdf/ which has that feature.
